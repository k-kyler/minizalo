### Upload image, video & document

The upload handler will only accept the following file extensions:

- All extensions of image
- All extensions of video
- Document & archive: `.doc`, `.docx`, `.xls`, `.xlsx`, `.txt`, `.pdf`, `.rar`, `.zip`
